# 2024/10/02 리스트 구현

리스트 삽입과 조회, 삭제의 원리와 구현하는 소스코드 짜는 것

리스트 삽입 → 데이터를 저장할 자리 있나 먼저 조회 해보고 저장할 자리 있으면 데이터를 저장하고 데이터의 수를 증가시킴.

리스트 삭제 → 삭제할 조건에 맞는 데이터의 위치를 확인하고 찾으면 바로 직전에 참조가 이뤄진(삭제할 조건에 맞는) 데이터를 삭제한다. → 데이터가 삭제되면 빈자리가 생기고 그 빈자리를 매꾸기 위해 뒤에 있던 데이터들의 위치를 앞으로 옮긴다.

리스트에 구조체 변수 저장하는 코드 있는데 c언어 쪽에서 부족한 부분이 있어서 해석이 잘 안됨. 필요한 앞 부분 복습하고 다시 봐야 할 듯.

이 뒷 부분도 같은 부분을 다시 공부해야 이해가 가능한데 오늘은 c언어 책이 없고 저녁에 일정이 있어서 여기까지.
